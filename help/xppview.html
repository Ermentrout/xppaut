View axes

This selects one of 2 different types of graphs: 2D or 3D or brings up a new animation window or a space-time array window

If you select the 2D curve, a dialog box will appear asking you for

  • the variables to place on the axes;
  • upper and lower limits of the two axes as in the window command; and
  • the labels for the axes

If you select 3D, a more complicated dialog box appears asking for:

  • the variables to place on the axes;
  • their max and min values;
  • 4 more numbers, XLO, etc. XPP first scales the data to fit into a cube with corners (-1,-1,-1) and (1,1,1). Rotation of this cube is performed and then projected into the two-D window. XLO, etc define the scales of this projection and are thus unrelated to the values of your data. Use the Window|Fit option to have XPP set these for you.
  • Axes labels
